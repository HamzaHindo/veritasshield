# 🛡️ Veritas Shield

**Veritas Shield** is an AI-powered legal assistant that helps users analyze contracts, detect risks, and identify conflicts across multiple agreements.

---

## 🚀 Overview

Legal documents are often complex and difficult to understand for non-experts. Most existing tools analyze contracts in isolation, missing the bigger picture.

Veritas Shield introduces a **personal legal intelligence system** that treats a user’s contracts as a connected system, enabling:

* Smarter risk detection
* Cross-contract conflict analysis
* Clear and visual insights

---

## ✨ Key Features

* 📄 **Contract Analysis**
  Upload contracts and receive AI-powered insights

* ⚠️ **Risk Detection & Scoring**
  Each clause is analyzed and assigned a risk score (1–10)

* 🔥 **Risk Heatmap**
  Visual representation of high-risk clauses

* 🔗 **Clash Engine**
  Detects conflicts between multiple contracts

* 🧠 **Knowledge Graph (Legal Memory)**
  Maintains context across all user agreements

* 🔍 **Visual Contract Diff**
  Highlights changes between contract versions

---

## 🧠 How It Works

1. User uploads a contract
2. The AI engine extracts and analyzes clauses
3. Risks are scored and categorized
4. Data is stored and linked to previous contracts
5. The system detects conflicts and provides insights
