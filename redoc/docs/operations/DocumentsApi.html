This is the API

documentClauses

Returns all extracted clauses for a specific document.

URL

http://localhost:8000/files/documents/{doc_id}/clauses/

HTTP Method

GET

Response Type

List

Parameters

  • Path: doc_id Integer

    Document ID

documentsDestroy

Deletes a document by ID. Requires authentication.

URL

http://localhost:8000/files/documents/{id}/

HTTP Method

DELETE

Response Type

Parameters

  • Path: id Integer

    Document ID

documentsList

Returns a list of all documents. Requires authentication.

URL

http://localhost:8000/files/documents/

HTTP Method

GET

Response Type

List

Parameters

documentsRetrieve

Returns details of a specific document by ID. Requires authentication.

URL

http://localhost:8000/files/documents/{id}/

HTTP Method

GET

Response Type

Document

Parameters

  • Path: id Integer

    Document ID

documentsUpdate

Updates an existing document. Requires authentication.

URL

http://localhost:8000/files/documents/{id}/

HTTP Method

PUT

Response Type

Document

Parameters

  • Form: file byte[]

  • Form: title String

  • Form: lang String

  • Form: file_extension String

  • Path: id Integer

    Document ID
