This is the API

authLogin

Authenticates a user with email and password, returns JWT token pair.

URL

http://localhost:8000/auth/login/

HTTP Method

POST

Response Type

TokenPair

Parameters

  • Body: body Auth_login_body(Auth_login_body)

authLogout

Blacklists the provided refresh token to log the user out.

URL

http://localhost:8000/auth/logout/

HTTP Method

POST

Response Type

inline_response_205

Parameters

  • Body: body Auth_logout_body(Auth_logout_body)

authRefresh

Returns a new access token using a valid refresh token.

URL

http://localhost:8000/auth/refresh/

HTTP Method

POST

Response Type

inline_response_200

Parameters

  • Body: body Auth_refresh_body(Auth_refresh_body)

authRegister

Creates a new user account and returns JWT token pair.

URL

http://localhost:8000/auth/register/

HTTP Method

POST

Response Type

TokenPair

Parameters

  • Body: body Auth_register_body(Auth_register_body)
