This is the API

analyzeDocument

Uploads a document, performs OCR to extract text, analyzes the content to extract clauses, finds similar clauses across existing documents, and detects conflicts.

URL

http://localhost:8000/analyze/

HTTP Method

POST

Response Type

AnalysisResult

Parameters

  • Form: file byte[]

  • Form: title String

  • Form: language String

analyzeSave

Saves/inserts the analysis results for a previously uploaded document. Fetches the document, validates its text content, and persists the analysis.

URL

http://localhost:8000/analyze/save/

HTTP Method

POST

Response Type

AnalysisResult

Parameters

  • Body: body Analyze_save_body(Analyze_save_body)
